THIẾT KẾ SEQUENCE DIAGRAM CHO CHỨC NĂNG VIEW BOOKING REQUEST (RECEPTIONIST)
DÀNH CHO VISUAL PARADIGM

--------------------------------------------------------------------------------
1. THÀNH PHẦN THAM GIA (LIFELINES/OBJECTS)
--------------------------------------------------------------------------------

- Actor: Receptionist (Lễ tân)
- Boundary (Giao diện): :Browser / Giao diện Dashboard (JSP)
- Controller (Điều khiển): :ReceptionistDashboardController
- Database Layer (DAO/Repository):
  * :BookingDAO
  * :RoomTypeRepository
- Database: Database (Cơ sở dữ liệu)

--------------------------------------------------------------------------------
2. KỊCH BẢN TUẦN TỰ (STEPS OF SEQUENCE DIAGRAM)
--------------------------------------------------------------------------------

1. Receptionist nhấn chọn tab "Bookings" hoặc nhập từ khóa tìm kiếm (keyword), chọn trạng thái lọc (status) trên trình duyệt.
2. :Browser gửi yêu cầu HTTP GET đến Servlet với các tham số:
   GET /receptionist/dashboard?tab=bookings&status={status}&keyword={keyword}
3. :ReceptionistDashboardController nhận yêu cầu và kiểm tra Session/Role của User.
   - [Alt: Nếu Session null hoặc Role không hợp lệ] -> Redirect về trang Login.
   - [Ngược lại] -> Tiếp tục xử lý:
4. :ReceptionistDashboardController gọi hàm nội bộ loadBookingTab(request) để lấy dữ liệu.
5. Trong loadBookingTab():
   - Controller khởi tạo đối tượng :BookingDAO.
   - Controller khởi tạo đối tượng :RoomTypeRepository.
6. Controller gọi hàm dao.getBookings(statusFilter, keyword) để truy vấn danh sách đặt phòng.
7. :BookingDAO thực hiện kết nối cơ sở dữ liệu và gửi truy vấn SQL SELECT tới Database.
8. Database trả về kết quả (ResultSet).
9. :BookingDAO map dữ liệu từ ResultSet sang danh sách List<Booking> và trả về cho Controller.
10. Controller gọi hàm roomTypeRepo.getAllRoomTypes() để lấy danh sách loại phòng hỗ trợ cập nhật thông tin trong modal.
11. :RoomTypeRepository truy vấn Database và nhận về danh sách loại phòng List<RoomTypeInfo>.
12. Controller gọi tiếp các hàm dao.countAll() và dao.countByStatus(status) để lấy số liệu thống kê nhanh hiển thị trên các badge.
13. :BookingDAO truy vấn đếm từ Database và trả về các số lượng (int).
14. Controller gọi request.setAttribute() để lưu các danh sách và số liệu vừa lấy được.
15. Controller gọi request.getRequestDispatcher(...).forward() để chuyển hướng (forward) yêu cầu và dữ liệu sang trang JSP (/WEB-INF/views/dashboard/receptionist.jsp).
16. JSP xử lý cấu trúc giao diện, hiển thị bảng danh sách booking, các nút lọc và trả về mã HTML hoàn chỉnh cho :Browser.
17. :Browser hiển thị danh sách Booking requests đã lọc cho Receptionist xem.

--------------------------------------------------------------------------------
3. MÃ NGUỒN SƠ ĐỒ PLANTUML (Để import trực tiếp vào Visual Paradigm hoặc vẽ trên PlantUML)
--------------------------------------------------------------------------------

@startuml
autonumber

actor "Receptionist\n(Lễ tân)" as Rec
participant ":Browser\n(Giao diện)" as Browser
participant ":ReceptionistDashboardController" as Ctrl
participant ":BookingDAO" as BookingDAO
participant ":RoomTypeRepository" as RoomRepo
database "Database" as DB

Rec -> Browser : Chọn bộ lọc status & nhập từ khóa tìm kiếm
Browser -> Ctrl : GET /receptionist/dashboard?tab=bookings&status={status}&keyword={keyword}
activate Ctrl

note over Ctrl : Kiểm tra Session và Quyền hạn\n(Role == RECEPTIONIST)

alt Không được phép (Session null hoặc sai Role)
    Ctrl --> Browser : Redirect to /home/login?error=unauthorized
else Hợp lệ
    Ctrl -> Ctrl : loadBookingTab(request)
    activate Ctrl
    
    create BookingDAO
    Ctrl -> BookingDAO : new BookingDAO()
    
    create RoomRepo
    Ctrl -> RoomRepo : new RoomTypeRepository()

    Ctrl -> BookingDAO : getBookings(statusFilter, keyword)
    activate BookingDAO
    BookingDAO -> DB : Execute SELECT Query (với status & keyword)
    activate DB
    DB --> BookingDAO : Trả về ResultSet
    deactivate DB
    BookingDAO --> Ctrl : Trả về List<Booking>
    deactivate BookingDAO

    Ctrl -> RoomRepo : getAllRoomTypes()
    activate RoomRepo
    RoomRepo -> DB : Execute SELECT Query (Room Types)
    activate DB
    DB --> RoomRepo : Trả về ResultSet
    deactivate DB
    RoomRepo --> Ctrl : Trả về List<RoomTypeInfo>
    deactivate RoomRepo

    Ctrl -> BookingDAO : countAll() / countByStatus()
    activate BookingDAO
    BookingDAO -> DB : Execute SELECT COUNT Query
    activate DB
    DB --> BookingDAO : Trả về kết quả
    deactivate DB
    BookingDAO --> Ctrl : Trả về số lượng (int)
    deactivate BookingDAO

    Ctrl -> Ctrl : setAttribute("bookingList", ...)\nsetAttribute("cntAll", ...)
    deactivate Ctrl
    
    Ctrl -> Browser : forward("/WEB-INF/views/dashboard/receptionist.jsp")
    deactivate Ctrl
    
    activate Browser
    Browser --> Rec : Hiển thị danh sách Booking Request cùng các bộ lọc
    deactivate Browser
end
@endum
