LUỒNG CODE NGHIỆP VỤ ĐẶT PHÒNG (ROOM BOOKING)

I. TỔNG QUAN LUỒNG THỰC THI (FLOW)
1. Người dùng (Customer) truy cập trang tạo đơn đặt phòng, điền form và gửi yêu cầu (POST /customer/booking/create).
2. Controller (CustomerBookingsController.java) nhận request, lấy các tham số từ form và tạo đối tượng Booking.
3. Controller gọi Service (BookingService.java) để kiểm tra logic (số lượng phòng trống, ngày hợp lệ) và tính toán tiền phòng.
4. Service gọi DAO (BookingDAO.java) để thực thi câu lệnh SQL INSERT vào database.
5. Nếu thành công, Controller chuyển hướng người dùng về trang lịch sử đặt phòng kèm thông báo thành công. Nếu lỗi, Controller giữ lại thông tin form và hiển thị thông báo lỗi.

II. GIẢI THÍCH CHI TIẾT TỪNG DÒNG CODE VÀ FILE LIÊN QUAN

1. File: CustomerBookingsController.java
Nhiệm vụ: Nhận request từ trình duyệt, phân luồng chức năng, trích xuất dữ liệu, và điều hướng trang (view).

Phương thức xử lý tạo Booking: handleCreateBooking(HttpServletRequest request, HttpServletResponse response, int accountId)

// Lấy dữ liệu từ form do người dùng nhập vào
String customerName = request.getParameter("customerName"); 
String phone = request.getParameter("phone");
String email = request.getParameter("email");
String checkInStr = request.getParameter("checkInDate");
String checkOutStr = request.getParameter("checkOutDate");
String note = request.getParameter("note");
String bookingType = request.getParameter("bookingType"); // Xác định là đặt "single" (1 loại phòng) hay "multi" (nhiều loại phòng)

Booking booking = new Booking(); // Tạo đối tượng Booking mới để lưu trữ dữ liệu
booking.setAccountId(accountId); // Set ID của tài khoản khách hàng đang đăng nhập
booking.setCustomerName(customerName); // Set tên khách
booking.setPhone(phone); // Set SĐT
booking.setEmail(email); // Set email
booking.setNote(note); // Set ghi chú/yêu cầu đặc biệt

// Parse ngày check-in, check-out từ chuỗi (String) sang kiểu Date (SQL)
if (checkInStr != null && !checkInStr.trim().isEmpty()) {
    booking.setCheckInDate(Date.valueOf(checkInStr));
}
if (checkOutStr != null && !checkOutStr.trim().isEmpty()) {
    booking.setCheckOutDate(Date.valueOf(checkOutStr));
}

// Xử lý trường hợp đặt NHIỀU loại phòng cùng lúc (Multi)
if ("multi".equalsIgnoreCase(bookingType)) {
    String[] roomTypeIds = request.getParameterValues("roomTypeId[]"); // Lấy danh sách ID loại phòng
    String[] roomQuantities = request.getParameterValues("roomQuantity[]"); // Lấy danh sách số lượng phòng tương ứng
    
    // Validate kiểm tra xem số lượng phòng trống có đủ cho tất cả lựa chọn hay không
    for (int i = 0; i < roomTypeIds.length; i++) {
        int rtId = Integer.parseInt(roomTypeIds[i].trim());
        int qty = Integer.parseInt(roomQuantities[i].trim());
        int available = bookingService.checkRoomAvailability(rtId, booking.getCheckInDate(), booking.getCheckOutDate());
        if (qty > available) {
            throw new Exception("MSG19"); // Báo lỗi nếu hết phòng trống
        }
    }

    // Tạo đơn đặt phòng CHA (Chứa loại phòng đầu tiên)
    int parentRoomTypeId = Integer.parseInt(roomTypeIds[0].trim());
    int parentQty = Integer.parseInt(roomQuantities[0].trim());
    booking.setRoomTypeId(parentRoomTypeId);
    booking.setRoomQuantity(parentQty);
    booking.setGroupBookingId(null); // Parent không có group ID
    
    bookingService.createBooking(booking); // Gọi Service lưu Parent booking
    int parentBookingId = booking.getBookingId(); // Lấy ID của Parent vừa tạo

    // Tạo các đơn đặt phòng CON (Chứa các loại phòng tiếp theo)
    for (int i = 1; i < roomTypeIds.length; i++) {
        Booking childBooking = new Booking(); // Tạo Booking con
        // Copy thông tin khách, ngày tháng từ form...
        childBooking.setGroupBookingId(parentBookingId); // Gán groupBookingId bằng ID của Parent để liên kết
        // Set roomTypeId và roomQuantity tương ứng cho Child
        bookingService.createBooking(childBooking); // Gọi Service lưu Child booking
    }
} else {
    // Xử lý trường hợp đặt MỘT loại phòng (Single)
    int roomTypeId = Integer.parseInt(request.getParameter("roomTypeId"));
    int quantity = Integer.parseInt(request.getParameter("roomQuantity"));
    
    booking.setRoomTypeId(roomTypeId);
    booking.setRoomQuantity(quantity);
    booking.setGroupBookingId(null); // Không thuộc nhóm nào

    bookingService.createBooking(booking); // Gọi service lưu Booking
}

response.sendRedirect(request.getContextPath() + "/customer/bookings?success=created"); // Chuyển hướng báo thành công

2. File: BookingService.java
Nhiệm vụ: Xử lý logic nghiệp vụ, tính toán, kiểm tra ràng buộc trước khi lưu vào database.

Phương thức: public boolean createBooking(Booking booking) throws Exception

// Kiểm tra xem Ngày trả phòng phải sau Ngày nhận phòng
Date checkIn = booking.getCheckInDate();
Date checkOut = booking.getCheckOutDate();
if (checkIn == null || checkOut == null || !checkIn.before(checkOut)) {
    throw new Exception("MSG17"); 
}

// Kiểm tra xem Note (Ghi chú) có vượt quá 500 ký tự hay không
if (booking.getNote() != null && booking.getNote().length() > 500) {
    throw new Exception("MSG03");
}

long nights = booking.getNights(); // Tính số đêm lưu trú (checkOut - checkIn)

// Lấy thông tin chi tiết Loại phòng để lấy Giá (Base Price)
RoomTypeInfo rt = roomTypeRepository.getRoomTypeById(booking.getRoomTypeId());
booking.setRoomTypeName(rt.getTypeName()); // Lưu lại tên loại phòng vào đối tượng Booking

// Kiểm tra số lượng phòng trống bằng cách: (Tổng số phòng của loại đó) - (Số phòng đã được đặt trong giai đoạn checkIn -> checkOut)
int totalRoomsInHotel = getRoomCountByTypeId(booking.getRoomTypeId());
int bookedRoomsCount = bookingDAO.getBookedRoomsCountForDates(booking.getRoomTypeId(), checkIn, checkOut);
int availableRooms = totalRoomsInHotel - bookedRoomsCount;

// So sánh nếu khách đặt nhiều hơn số phòng còn trống thì ném ra lỗi (MSG19)
int qty = booking.getRoomQuantity() > 0 ? booking.getRoomQuantity() : 1;
if (qty > availableRooms) {
    throw new Exception("MSG19"); 
}

// Tính tổng tiền phòng: Giá gốc * Số lượng phòng * Số đêm
double totalAmount = rt.getBasePrice() * qty * nights;
booking.setTotalAmount(totalAmount); // Set tổng tiền vào Booking
booking.setStatus("Pending"); // Mặc định đơn đặt phòng mới có trạng thái là Pending (Chờ duyệt/Chờ nhận phòng)

// Gọi DAO để INSERT xuống database
int bookingId = bookingDAO.createBooking(booking);
booking.setBookingId(bookingId); // Cập nhật lại ID tự tăng (Identity) vừa sinh ra vào Object Booking
return true;

3. File: BookingDAO.java
Nhiệm vụ: Giao tiếp với cơ sở dữ liệu (Database), thực hiện câu lệnh SQL INSERT.

Phương thức: public int createBooking(Booking b)

// Chuẩn bị câu lệnh SQL INSERT (Sử dụng bảng dbo.Booking)
String sql = "INSERT INTO dbo.Booking (account_id, customer_name, phone, email, room_type_id, room_quantity, check_in_date, check_out_date, total_amount, status, note, group_booking_id, created_at, updated_at) "
        + "VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, SYSDATETIME(), SYSDATETIME())";

// Mở kết nối đến SQL Server
try (Connection conn = DBContext.getConnection()) {
    useDatabase(conn); // Đảm bảo sử dụng đúng Database HotelManagementDB
    
    // PrepareStatement với cờ Statement.RETURN_GENERATED_KEYS để lấy ID tự động tăng sau khi insert
    try (PreparedStatement ps = conn.prepareStatement(sql, Statement.RETURN_GENERATED_KEYS)) {
        
        // Truyền lần lượt các tham số vào dấu chấm hỏi (?)
        ps.setInt(1, b.getAccountId());
        ps.setString(2, b.getCustomerName());
        ps.setString(3, b.getPhone());
        ps.setString(4, b.getEmail());
        ps.setInt(5, b.getRoomTypeId());
        ps.setInt(6, b.getRoomQuantity());
        ps.setDate(7, b.getCheckInDate());
        ps.setDate(8, b.getCheckOutDate());
        ps.setDouble(9, b.getTotalAmount());
        ps.setString(10, b.getStatus() != null ? b.getStatus() : "Pending");
        ps.setString(11, b.getNote() != null ? b.getNote().trim() : "");
        
        // Xử lý khóa ngoại Nullable (group_booking_id)
        if (b.getGroupBookingId() != null) {
            ps.setInt(12, b.getGroupBookingId());
        } else {
            ps.setNull(12, Types.INTEGER); // Gán NULL nếu là Single Booking hoặc là Parent Booking
        }

        // Thực thi câu lệnh SQL (Trả về số row bị ảnh hưởng)
        int affected = ps.executeUpdate();
        
        // Nếu insert thành công (affected > 0), lấy ra ID (booking_id) vừa được tạo
        if (affected > 0) {
            try (ResultSet rs = ps.getGeneratedKeys()) {
                if (rs.next()) {
                    return rs.getInt(1); // Trả về ID tự động tăng
                }
            }
        }
    }
}
return -1; // Trả về -1 nếu thất bại
