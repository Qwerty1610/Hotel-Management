================================================================================
DESCRIPTION OF RELATIONSHIPS:
1. Account (1) -------- (0..*) Booking
   - A registered account can place multiple bookings.
   
2. RoomType (1) ------- (0..*) Booking
   - Each booking is reserved for a single RoomType.
   
3. RoomType (1) ------- (0..*) Room
   - Each physical room belongs to one RoomType category.
   
4. Booking (1) -------- (0..*) RoomAssignment (0..*) -------- (1) Room
   - A many-to-many bridge table that assigns specific physical rooms to a booking
     at check-in time.
